# Second Brain

Eine schlanke Notizen-Web-App: Notizen anlegen, taggen, durchsuchen und filtern – alles direkt im Browser gespeichert (kein Backend nötig).

Gebaut in Zusammenarbeit mit Claude (Anthropic) als iteratives KI-unterstütztes Projekt.

## Features

- Notizen mit Titel, Inhalt und Tags anlegen
- Volltextsuche über Titel, Inhalt und Tags
- Nach Tags filtern (Klick auf einen Tag-Chip)
- Notizen löschen
- Daten bleiben lokal im Browser gespeichert (`localStorage`)

## Nutzung

Einfach `index.html` im Browser öffnen – kein Build-Schritt, keine Installation nötig.
